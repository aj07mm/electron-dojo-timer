# Electron Dojo Timer

This project is for make DOJO with JavaScript.

## Installation

_This project use yarn_

1. clone
2. `yarn`
3. `yarn run dev`


# problems with node-gyp:

rm -rf package-lock.json
rm -rf node_modules
rm -rf ~/.node-gyp
yarn install
